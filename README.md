# Istanbul Traffic Density Clustering Analysis 

This project analyzes traffic density patterns in Istanbul using machine learning (K-Means) and visualizes the results on an interactive map.

## Features
- **K-Means Clustering:** Groups traffic data into density zones.
- **PCA Visualization:** 2D scatter plot of traffic clusters.
- **Interactive Map:** Folium-based HTML map showing hotspots.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Add your `traffic_density_202501.csv` to the root folder.
3. Run: `python main.py`

## Results
The analysis generates an interactive map named `istanbul_trafik_haritasi.html`.
